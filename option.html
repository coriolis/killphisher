Security Image  lock

Your security image is flashed in a popup when you visit protected pages. Look for your secure image whenever you login to Google, Facebook, Amazon, Paypal and other pages you have protected. If you do not see it, double check the URL.
Your security image should be known only to you.

Image Library

Protected Pages

Login pages on which you will see your personal security image. Web pages you browse are scanned for similarity with protected pages, and you will be alerted to possible phishing attempts.

Safe Domains

Trusted domains which are highly unlikely to host phishing pages. Pages on these sites are not scanned to reduce overhead.

For example, if we add google.com to the list, all pages hosted by google.com, mail.google.com and other subdomains are exempt from scanning. This reduces performance overhead for 90% browsing activity, while still protecting the 10% which really needs it.

About

SpotPhish is a Chrome extension which protects against zero-day phishing by creating large visual differences between legitimate sites and phishing pages which attempt to mimic them. This is done in the following manner:

  • Visiting a genuine page triggers a popup with a personal security image, which is displayed for a few seconds.
  • The extension takes screenshots of the active browser tab and raises an alarm if the current page visually resembles a protected page, but belongs to an unknown domain.
